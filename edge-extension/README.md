# Eva Wolf Canvas Helper (Edge Extension)

This extension stays local on your machine. You do **not** need GitHub to use it.

## Features
- Small wolf logo in the popup.
- **Option A:** one-click copy of Canvas discussion text to clipboard.
- Highlights and clickable links to threads that still need Eva's reply.

## Install in Microsoft Edge (local only)
1. Open Edge and go to `edge://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this `edge-extension` folder (the folder containing `manifest.json`).

## Use
1. Open a Canvas discussion page.
2. Click the extension icon.
3. Click **Copy discussion text (Option A)** and paste into your Colab/Notebook input cell.
4. Click **Highlight + link missing threads** to open a right-side navigator panel.

## Update later
- Replace files in this folder, then click **Reload** on the extension card in `edge://extensions`.
